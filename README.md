# 🌍 Travel Planning & Documentation

A Jekyll-based site for documenting and sharing travel itineraries. Plan trips, organize daily activities, and create shareable travel guides.

## 📂 Project Structure

```
├── index.md                           # Homepage listing all trips
├── _config.yml                        # Jekyll configuration
├── _layouts/
│   └── default.html                   # Site template
└── Trips/
    └── Yellowstone/
        ├── index.md                   # Trip overview
        ├── DailyActivity.md           # Day-by-day itinerary
        └── Todo.md                    # Bookings & prep checklist
```

## 🚀 Getting Started

### Local Development
```bash
cd homework
gem install jekyll bundler
jekyll serve
```
Then visit `http://localhost:4000`

### Deploy to GitHub Pages
Push to your repository and enable GitHub Pages in Settings. Your site will be live at:
`https://madathil22.github.io/homework/`

## ✨ Features

- 📝 Organize trips with multiple pages
- ✅ Checklists for bookings and prep
- 📅 Daily activity itineraries
- 🎨 Styled with modern design
